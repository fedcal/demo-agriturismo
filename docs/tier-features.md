# Funzionalità per Tier — Agriturismo

Tre livelli di template per agriturismo e farm-to-table, dal sito vetrina al full platform con tracciamento filiera blockchain-light, wedding planner e cooking class marketplace.

## Tier Base — €600-900 (consegna 2-3 settimane)

**Per chi**: Agriturismo che vuole presence online autentica e attirare turista consapevole.  
**Sforzo stimato**: ~90h.

### Funzionalità incluse

- **Home Hero** paesaggio rurale + prodotti (oliveto, vigneto, orto)
- **Galleria Produzione**
  - Foto olio/vino/formaggio con descrizione
  - Varietà (Frantoio, Coratina) + raccolta mese
  - Contesto rurale: foto campi, fattoria, famiglia

- **Camere Strutturate**
  - Casale, Cottage, Glamping (con foto)
  - Posti letto, amenity (riscaldamento, acqua calda)
  - Prezzo/notte generico

- **Attività Farm-to-Table**
  - Wine tasting in cantina
  - Cooking class + cena (olio/vino made-in-farm)
  - Truffle hunt + guide locale
  - Horseback trekking paesaggio

- **Farm-to-Table Menu**
  - Piatti stagionali (primavera vs estate vs autunno)
  - Ingredienti tracciati: "Olio frantoio dicembre", "Vino Barbera 2020"
  - Photo food beautiful

- **Story Famiglia**
  - Bio fondatori (nonna che fa pasta fresca)
  - Tradizione generazioni
  - Filosofia naturale/biologico/sostenibile

- **Schema LodgingBusiness + Place** JSON-LD
- **Mobile-first responsive**

### Cosa NON è incluso

- Booking attività
- Tracciamento filiera
- Wedding planning
- Cooking class Zoom monetizzato
- Multi-lingua

| Hosting + SSL | €60/anno | Image-heavy agriturismo |

## Tier Intermedio — €1.800-2.500 (consegna 4-6 settimane)

**Per chi**: Agriturismo con esperienza (wine tasting, cooking class) che vuole booking e sourcing transparency.  
**Sforzo stimato**: ~300h.

### Funzionalità incluse (oltre al Base)

- **Booking Sistema Integrato**
  - Calendar camere realtime (avoid overbooking)
  - Booking attività: wine tasting 15/03 ore 15:00 (max 8 persone)
  - Cooking class booking (data, menu choice, pre-arrival dieta note)
  - Horsetracking trekking time slot
  - Stripe caparra 30%

- **Experience Packages (Bundle)**
  - "Weekend romantico 2 notti + wine tasting + cooking dinner" (€450 per coppia)
  - "Family farm immersion" (camere + horse + truffle hunt + cooking)
  - Pre-built templates cliente sceglie

- **Sourcing Transparency**
  - Olio: varietà + epoca raccolta + note gustative
  - Vino: cantina + vitigno + anno + abbinamenti food
  - Ingredienti dish: traccia sourcing (olio km0, mozzarella fornitore locale)
  - QR code piatto → storia ingrediente (blockchainy senza blockchain)

- **Newsletter Stagionale**
  - Primavera: "Nuova raccolta asparagi, ricetta zuppa"
  - Estate: "Wine & music festival 20/7"
  - Autunno: "Truffle season booking aperto"

- **Multi-lingua IT/EN/DE/FR**
  - Attrae turista europeo/USA (agriturismo Instagram-famous)

- **Google Business Profile Auto-Sync**

## Tier Avanzato — €5.000-8.000 (consegna 10-12 settimane)

**Per chi**: Agriturismo multi-proprietà con wedding venue, cooking class monetizzato (Zoom remote), filiera blockchain-light, content hub SEO.  
**Sforzo stimato**: ~580h.

### Funzionalità incluse (oltre all'Intermedio)

- **Filiera Tracking QR (Blockchain-light)**
  - Olio: QR code bottiglia → storia lotto (data raccolta, alberi, lavorazione)
  - Vino: QR → vitigno, fermentazione, anno, abbinamenti
  - Formaggio: QR → latte fornitore, caseificio, stagionatura
  - Photo + blockchain hash (no blockchain expense, just versioning)
  - Certificazione DOP/IGP evidenziata

- **Wedding Venue Planner Configurator**
  - Client seleziona guest count (30/50/75/100)
  - Auto-suggest: tavoli configuration, menu options, wine list
  - Activity bundle: "Cerimonia + colazione + cooking class + overnight"
  - Pricing dinamico guest count
  - PDF proposal + contract e-sign

- **Cooking Class Live Zoom (Monetizzabile)**
  - Proprietario insegna: ragu ragù dalla family recipe
  - Live sett mercoledì 18:00 CET (target turista remoti USA)
  - Recording venduto (€15 video on-demand per 30 giorni)
  - Booking ticketing integrato
  - Supply list email pre-class (client compa ingredienti)

- **Wellness Retreat Packages**
  - Yoga + breakfast + wine tasting (3 giorni €380)
  - Spa day + cooking + overnight
  - Detox retreat: juice + yoga + meditation + farm work
  - Pre-built templates + custom options

- **Virtual Farm Tour 360° + Vlog**
  - Matterport: cantina, orchards, cucina farm
  - Vlog YouTube: "Giorno nella vita agriturismo" (4-6 min, monthlyl)
  - Instagram Reels: process video (truffle hunt, harvest, cooking)
  - TikTok auto-repurpose trending clips

- **Content Hub Blog SEO (52 articoli/anno)**
  - "Farm-to-table ricette italiano"
  - "Wine tasting guide Piemonte"
  - "Organic farming sostenibilità"
  - Long-tail SEO: "agriturismo weekend romantico toscana"

- **Multi-Location Management** (2-3 proprietà)
  - Consolidated dashboard: occupancy, revenue, activities
  - Cross-property loyalty (visitor stays Milano farm → sconto Roma farm)

- **AI Chef Assistant** (RAG on-prem Ollama llama3.1:8b)
  - Knowledge base: recipe family, wine pairing, allergie handling
  - Query: "Cliente vegan, suggerisci menu festa"
  - Marketing assist: email suggerimento package stagionale
